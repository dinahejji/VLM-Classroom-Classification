VLM Classroom Classification

Vision-Language Model for Classroom Type and Topic Prediction

üß† About the Project

This project fine-tunes a Vision-Language Model (VLM) to classify classroom scenes based on images and transcripts. It supports multiple types such as: Normal Lecture, Flipped Classroom, Lab Lecture, Active Learning, Hybrid Sessions, and Exams. It also infers academic topics like Physics, Chemistry, AI, or Sociology from student-teacher interaction.
